# Bank Marketing Term Deposit Prediction

## Objective
Predict whether a customer will subscribe to a term deposit using machine learning classification models.

## Dataset
UCI Bank Marketing Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- SHAP

## Models Used
- Logistic Regression
- Random Forest Classifier

## Evaluation Metrics
- F1 Score
- Confusion Matrix
- ROC-AUC Score
- ROC Curve

## Key Insights
- Dataset is highly imbalanced
- Customer behavior varies by job, education, and contact month
- Random Forest performed better for non-linear patterns
- SHAP helped explain prediction behavior
