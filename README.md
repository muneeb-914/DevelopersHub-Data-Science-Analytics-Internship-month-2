## Internship Overview

This repository contains the completed Data Science & Analytics internship tasks assigned by DevelopersHub Corporation.

The internship tasks cover practical machine learning and analytics problems including classification, clustering, time series forecasting, cost-based risk optimization, explainable AI, and business intelligence dashboarding.

Each task is placed in a separate folder with its own notebook/code, README file, requirements, and related outputs.

---

## Completed Tasks

### Task 1: Bank Marketing Term Deposit Prediction

**Objective:**  
Predict whether a bank customer will subscribe to a term deposit after a marketing campaign.

**Key Work:**
- Exploratory Data Analysis
- Categorical feature encoding
- Logistic Regression
- Random Forest Classifier
- Confusion Matrix, F1 Score, ROC Curve
- SHAP explainability

---

### Task 2: Customer Segmentation using K-Means

**Objective:**  
Segment customers based on income and spending behavior.

**Key Work:**
- EDA
- K-Means Clustering
- Elbow Method
- PCA Visualization
- Marketing strategy recommendations

---

### Task 3: Energy Consumption Time Series Forecasting

**Objective:**  
Forecast short-term household energy consumption using historical time-based patterns.

**Key Work:**
- Time series parsing and resampling
- Feature engineering
- ARIMA
- Prophet
- XGBoost
- MAE and RMSE evaluation
- Actual vs forecast visualization

---

### Task 4: Loan Default Risk with Cost Optimization

**Objective:**  
Predict loan default risk and optimize the decision threshold using business cost analysis.

**Key Work:**
- Data preprocessing
- Classification modeling
- Threshold tuning
- False positive and false negative cost analysis
- Business cost minimization

---

### Task 5: Global Superstore Streamlit Dashboard

**Objective:**  
Build an interactive business dashboard for analyzing sales, profit, customers, and segment performance.

**Key Work:**
- Streamlit dashboard
- Interactive filters
- KPI cards
- Sales and profit analysis
- Top customers
- Regional and category-level insights
